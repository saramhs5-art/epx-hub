📡 시트 연동 API URL
대치점 일정표 스프레드시트에 Code.gs를 설치하고 배포한 후 URL을 입력하세요.
입력하면 모든 매니저가 실시간으로 같은 일정을 공유합니다.
🔄 자동 새로고침
주기 시트 변경사항을 자동으로 불러옵니다
📋 설치 가이드
1대치점 일정표 스프레드시트 열기
2상단 메뉴 → 확장 프로그램 → Apps Script
3Code.gs 파일 내용 전체 복사 → 편집기에 붙여넣기
4상단 배포 → 새 배포 → 유형: 웹 앱
5액세스: 모든 사용자 → 배포 → URL 복사
6위 입력창에 붙여넣기 → 저장 & 연결
7일정표 스프레드시트에 허브연동 탭이 자동 생성됨
8허브연동 탭에 날짜 / 내용 / 타입 / 지점 입력 → 허브 즉시 반영
